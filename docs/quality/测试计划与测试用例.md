**测试计划与测试用例.md**

**文档版本**：1.0  
**更新日期**：2026-03-24  
**项目名称**：EroticaForge - 本地 NSFW 角色扮演小说生成工具

### 1. 测试目标
- 确保系统在 GTX 1660 6GB + 16GB RAM 环境下稳定运行
- 验证长篇 NSFW 角色扮演小说生成的一致性、NSFW 服从度、防重复能力
- 验证 RAG 记忆准确性、StoryState 维护正确性
- 保证生成内容质量达到可实际创作水平

### 2. 测试范围

**纳入测试**：
- 核心功能流程（上传文档 → RAG 检索 → 生成 → 后处理）
- 单模型模式与多模型协作模式
- 防死循环 / 防重复机制
- PostgreSQL（pgvector）持久化（重启后记忆是否保留）
- 流式输出稳定性
- 极端 NSFW 题材服从度

**暂不测试**：
- 多用户并发（当前为单机工具）
- 图像生文功能（后续版本）

### 3. 测试环境

- **硬件**：GTX 1660 6GB + 16GB RAM（主测） / GTX 1050 Ti 4GB（兼容性测试）
- **软件**：Windows 11、JDK 21、Ollama（bge-m3）、PostgreSQL（pgvector）、llama-server、Maven
- **模型组合**：
  - 主模型：`vaultbox/qwen3.5-uncensored:9b` (Q4_K_M)
  - 总结模型：`qwen3.5:4b`
  - 嵌入模型：`bge-m3`（Ollama）

### 4. 测试类型与策略

| 测试类型         | 描述                                   | 执行方式             |
|------------------|----------------------------------------|----------------------|
