### 1. 简介
EroticaForge 是一款**完全本地运行**的互动式 NSFW 角色扮演小说创作工具。

它的核心玩法是：
- 先制作**人物卡**和**世界观设定**
- 以**对话 + 场景描述**的方式与角色互动
- 系统以**小说形式**（第三人称叙述 + 对话气泡 + 内心独白）进行回复
- 生成过程中可以看到不同人物以**对话框气泡**的形式弹出文字

支持长篇记忆、极致 NSFW 自由、无审查、无联网，隐私完全由你掌控。

### 2. 启动系统
1. 按照《安装部署指南.md》完成 PostgreSQL（pgvector）、Ollama（bge-m3）、llama-server 与项目启动
2. 浏览器访问：**http://localhost:8080**
3. 点击「新建故事」开始使用

### 3. 核心操作流程

#### 3.1 创建新故事
1. 点击首页「新建故事」
2. 输入故事标题（如《黑丝人妻的秘密调教》）
3. 可添加标签（NTR、调教、黑丝等）
4. 点击创建，进入写作主界面

#### 3.2 准备人物卡与世界观（强烈建议先完成）
在左侧「文档管理」区域上传或新建以下内容：

- **人物卡**（最重要）：每个主要角色建立一张卡片，包含姓名、年龄、外貌、性格、性癖、说话风格、黑历史等
- **世界观设定 / 大纲**
- **已有章节总结**（后续可不断补充）

上传后，系统会自动将这些内容存入 RAG 记忆库，后续生成时会自动调用。

#### 3.3 开始写作（互动角色扮演模式）

这是本工具最推荐的使用方式：

1. **输入你的行动或对话**
   在右侧输入框中输入内容，支持以下几种方式：

   - **描述场景与动作**（最常用）：
     “我推开办公室的门，走到林晓曼身后，把手放在她的黑丝大腿上”

   - **直接与角色对话**：
     “对林晓曼说：今晚加班到这么晚，不如我送你回家吧？”

   - **混合输入**：
     “我突然从后面抱住她 [OOC: 重点描写触感和她的心理变化]”

   - **使用 OOC 指令控制生成**（非常实用）：
     - “[OOC: 不要重复上一段，立刻进入高潮部分]”
     - “[OOC: 加强感官描写和内心独白]”
     - “[OOC: 切换到女主第一人称视角]”

2. **点击「生成」或按 Ctrl + Enter**
   - 系统开始实时流式生成内容（文字像打字一样逐字出现）
   - **单次生成长度**：默认控制在 **450～650 字**（一段完整场景）

3. **生成内容的显示形式**
   生成结果会以**小说 + 视觉小说**混合样式呈现：

   - **小说旁白**（第三人称叙述）：正常段落显示
   - **人物对话**：以该角色的**对话气泡**形式弹出（不同角色使用不同颜色边框和头像）
   - **人物内心独白**：用斜体 + 浅色背景显示
   - **重要感官/动作描写**：使用 **粗体** 突出

   **示例显示效果**：
   ```
   我轻轻推开门走了进去。

   【林晓曼】（脸颊泛红，声音微微颤抖）：
   “主管……这么晚了，您找我有什么事吗？”

   *天哪，他离我好近……为什么心跳得这么厉害……*

   **他的手指顺着黑丝缓缓向上滑动，那种丝滑又带着压迫感的触感瞬间让我双腿发软。**
   ```

4. **生成完成后可进行的操作**
   - **继续互动**：直接在输入框输入下一句行动或对话
   - **手动编辑**：点击输出区域任意位置即可编辑文字
   - **保存为章节**：把当前这一轮互动保存为正式章节（推荐每 3～5 轮保存一次）
   - **重新生成**：只重新生成最后一次回复
   - **插入到指定位置**：可把当前内容插入到已有章节中

### 4. 写作小贴士

- 前期请尽量完善人物卡，人物卡越详细，后续一致性越好。
- 每写完 4～6 轮互动，建议上传一次「章节总结」，帮助系统更好地记忆剧情。
- 多使用 OOC 指令来控制节奏、避免重复、加强特定描写。
- 如果出现角色 OOC 或剧情重复，可输入 “[OOC: 重置当前场景，严格按照人物卡性格来]” 进行纠正。

### 5. 快捷键

- Ctrl + Enter → 生成 / 发送
- Ctrl + S → 保存当前章节
- Ctrl + R → 重新生成最后一次回复
- Esc → 停止生成

### 6. 常见问题

**Q：生成的内容总是重复怎么办？**  
A：开启「多模型模式」 + 使用 OOC 指令「不要重复上一段」

**Q：角色性格突然崩了？**  
A：检查人物卡是否完善，并多使用 OOC 指令纠正

**Q：想切换第一人称或第三人称？**  
A：使用 OOC 指令 “[OOC: 切换到女主第一人称]”
